les étapes d'une page web sont:
  1. Conception : création de la maquette et définition du contenu
  2. Développement : codage en HTML, CSS et JavaScript
  3. Intégration : transformation de la maquette en page fonctionnelle
  4. Test : vérification du bon fonctionnement (responsive, liens, bugs)
  5. Déploiement : mise en ligne de la page sur le web
les outils d'un projet web sont:
  • HTML
  • CSS
  • JavaScript
    • jQuery
    • React
  • API
HTML
Langage de balisage utilisé pour structurer le contenu d'une page web.
CSS
Langage de style qui permet de mettre en forme les éléments HTML (couleurs, tailles, disposition, etc.).
JavaScript
Langage de programmation qui permet de rendre les pages web interactives (animations, formulaires dynamiques, etc.).
